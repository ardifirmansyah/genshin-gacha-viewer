Genshin Gacha Viewer

How to use it?

  • Open your Genshin Impact App and navigate to wish -> history for once.
  • Navigate to %USERPROFILE%\AppData\LocalLow\miHoYo\Genshin Impact.
  • Open File output_log.txt.
  • Search for gacha/index.html in text file.
  • Copy all of the URL starting with https://webstatic-sea.mihoyo.com and end with #/log.
